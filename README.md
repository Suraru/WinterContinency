# WinterContinency
Winter Contingency is a total conversion mod to bring Bungie's Halo Universe into Arma

Created by Suraru



See the status document here: https://docs.google.com/document/d/1dP1tYLBxac20s9Xxf1V9OXuseawTt7BCWPlikU_CDMI
